# AWGDAS-v2

문서 작업 끝나면 본 파일 전체 교체 예정
지금은 초기 아이디어만 정리해둠

## 아이디어 / 브레인스토밍

- v1을 스프링 기반으로 마이그레이션하고, 이것저것 기능을 더 붙여서 배포하는 방향 생각 중
- 최우선 목표는 하네스 엔지니어링을 이용한 멀티세션 에이전틱 프로그래밍 연습
- v1 때 했던 것처럼 코드로 관리하는 상태 전이를 이번에도 가져갈지 고민 필요
  - 예전처럼 제한된 자유도로 상태 전이를 처리하게 할지
  - 아니면 에이전트가 다음 타자 에이전트를 직접 결정하게 할지
- 기능을 좀 더 넣어서 관리자 성격도 같이 가져갈지 고민 중
  - 대시보드
  - 생성 이력
  - 생성 결과 바로 실행
  - 생성 결과 다운로드
  - 생성 결과 수정
- 자바로 할지 코틀린으로 할지도 아직 고민 중
  - 지금 시점에서는 굳이 코틀린이어야 하나 싶기도 한데 좀 더 생각 필요
- 프론트도 뭘로 처리할지 고민
  - 여기에 리액트를 넣기엔 조금 과한 것 같기도 하고
  - 그냥 HTML 위주로 갈 수도 있고
  - 아니면 타임리프를 쓰는 것도 가능
- AWS에 EC2 하나 파서 실제 배포까지 가는 것도 고려 중
  - 테스트 가능한 계정 처리도 같이 넣고
  - LLM 인증 토큰은 내 것을 쓰되, 계정별 사용 횟수 제한을 두는 방식도 가능할 듯
